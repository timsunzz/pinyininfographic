# 儿童识字小报生成器

一个基于 Google Nano Banana Pro API 的儿童识字小报生成工具，帮助家长和老师为5-9岁儿童创建有趣的识字学习材料。

## 功能特点

- 🎨 **智能生成**：基于输入的主题和标题，自动生成包含拼音和汉字的识字小报
- 📚 **丰富词汇库**：内置20个常见场景（超市、医院、公园、学校等）的词汇数据库
- 🔄 **即时生成**：使用先进的AI图像生成技术，快速生成高质量小报
- 💾 **本地存储**：API密钥和历史记录安全保存在本地浏览器中
- 📱 **响应式设计**：支持电脑和移动设备使用

## 使用方法

### 1. 获取API密钥

1. 访问 [Replicate](https://replicate.com/) 网站并注册账号
2. 在账户设置中获取您的 API Key
3. 将API Key输入到应用的配置区域

### 2. 生成识字小报

1. **选择场景**：
   - 点击预设的场景按钮（如"超市"、"医院"等）
   - 或在输入框中输入自定义主题

2. **输入标题**：
   - 输入小报的大标题（如"走进超市"、"快乐医院"）
   - 系统会根据选择的主题自动生成默认标题

3. **生成小报**：
   - 点击"生成识字小报"按钮
   - 等待AI生成完成（通常需要30-60秒）

4. **查看和下载**：
   - 查看生成的小报图片和词汇清单
   - 点击"下载图片"保存小报
   - 或点击"重新生成"获取不同的版本

### 3. 查看历史记录

- 底部的"历史记录"区域会显示之前生成的小报
- 点击任何历史记录可以重新查看

## 技术栈

- **前端**：HTML5 + CSS3 + JavaScript (ES6+)
- **AI服务**：Google Nano Banana Pro (Replicate API)
- **存储**：浏览器本地存储 (LocalStorage)

## 场景词汇库

系统内置了20个常见场景，每个场景包含：
- **核心角色与设施**（3-5个词汇）
- **常见物品/工具**（5-8个词汇）
- **环境与装饰**（3-5个词汇）

当前支持的场景：
- 超市、医院、公园、学校、动物园
- 消防局、邮局、图书馆、博物馆、火车站
- 餐厅、理发店、玩具店、水果店、面包店
- 药店、银行、电影院、游泳池、操场

   - 请勿分享您的API密钥

## 注意事项

1. **生成时间**：每张小报的生成需要30-60秒，请耐心等待
2. **API费用**：使用Replicate API会产生费用，请注意使用量
3. **网络要求**：需要稳定的网络连接以调用API
4. **浏览器兼容**：推荐使用Chrome、Firefox等现代浏览器

## 故障排除

### 生成失败
- 检查API密钥是否正确
- 确认网络连接正常
- 检查Replicate账户余额

### 图片显示问题
- 刷新页面重试
- 清除浏览器缓存
- 检查图片URL是否有效
